# 🚀 Getting started with Tensorflow.js

### `init`

```
npm i --force
# or
yarn --force
```

### `develop`

```
npm run develop
# or
yarn develop
```

### `start`

```
npm run start
# or
yarn start
```

### `build`

```
npm run build
# or
yarn build
```
